# Where to Find the Migration Button

## Location

The **Firebase Migration** button is located in the **sidebar** (drawer menu).

### Steps to Find It:

1. **Open the sidebar** by clicking the hamburger menu (☰) icon in the top-left corner of the app

2. **Scroll down** to find the **"Pages"** expandable section

3. **Expand the "Pages" section** if it's not already expanded

4. You'll see these options in order:
   - 📝 **Developer Notes**
   - ☁️ **Firebase Migration** ← **This is the button you need!**
   - ─── (Divider)
   - 📱 Documentation screens (Home Screen, Checkout, Cart, etc.)

5. **Click on "Firebase Migration"** to open the migration screen

## What Happens When You Click It

1. Opens the **Migration Screen**
2. Shows options to:
   - **Migrate All** - Migrates screens and pages
   - **Screens Only** - Migrates only screen documentation
   - **Pages Only** - Migrates only API pages
3. Shows progress logs in real-time
4. **Note**: Images are NOT uploaded (uses local assets to avoid billing)

## Quick Access

The migration button has a **cloud upload icon** (☁️) next to the text "Firebase Migration" to make it easy to identify.
